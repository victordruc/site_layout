Срок реализации: 5 - 7 дней

От чего зависит стоимость разработки:
  1. Количества дизайн концептов
  2. Размера этикетки/ упаковки
  3. количества размеров, под которые должен быть адаптирован дизайн
  4. необходимости профессиональной фотосессии продукта
Материалы необходимые для создания дизайна этикетки/ упаковки
  1. Заполненный бриф на разработку этикетки/ упаковки
  2. логотип
  3. фирменный стиль/ брендбук - при наличие
  4. размеры этикетки/ упаковки (развертка - при наличие)
  5. текст для этикетки (название продукта, содержание, состав, описание, информация о производителе и тд)
  6. любая другая информация, которая может понадоб иться при разработке продукта (штрих код, перечень торговых символов и знаков и тд)
Финальная стоимость разработки оценивается по результатам личной встречи.

Файлы, которые предоставляются клиенту

Векторные файлы
(cdr, ai, eps, pdf)
Данные файлы пигодны для разработки любых рекламных носителях, любых размеров. При любом масштабировании, качество изображения не меняется.
Эти форматы можно открыть только с помощью специальных программ, в которых возможно дальнейшее редактирвание исходных файлов.
Растровые файлы
(png, jpeg)
Форматы данных файлов можно вставлять в любые электронные документы, размещать на сайтах, в соц сетях и тд. Они не являются исходником. При масштабировании качество изображения ухудшается.
